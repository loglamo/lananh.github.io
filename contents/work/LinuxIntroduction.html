Introduction of Linux

Linux

This is type of opensource operating systems with Linux core created by Linus Torvald. After many years of development, there are lots of versions of Linux (Distro called too) such as Ubuntu, Fedora, Debian, Kali,etc.

The reasons for using Linux

  • About copyright and cost: Linux is an opensource operating system, so users can use it free or cheap fee
  • This is a flexible operating system: Users can modify, fix as long as possibly; besides, Linux can be compatible with lots of environments such as servers, desktops, embeded systems
  • High security:Linux has a access permission rule system to every type of users. The root has the highest access permission in system. Other users have limited access permission. When the operating system has some bugs, they can be fixed by big community of Linux users.

Standard of the system hierarchy

Folder system in Linux is structured as the hierarchical tree. The highest folder is the root folder, as "/". All other folders are below the root folder. These folders are important system folders.

Terminal

Linux has a wonderful command line tool. It is like cmd in windows.

It support helpful, clear commands that are really friendly to users. You use "Ctrl+Alt+T" to open a new terminal or find terminal in find tool.
